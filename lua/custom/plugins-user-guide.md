# Plugins User Guide

## Git

### diffview.nvim

A unified, tabbed interface for viewing Git diffs and file history. Complements vim-fugitive by providing a dedicated diff viewer.

| Keymap | Description |
|--------|-------------|
| `<leader>gd` | Open diffview |
| `<leader>gh` | File history for current file |
| `<leader>gH` | Repo-wide file history |
| `<leader>gx` | Close diffview |
